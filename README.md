# ML Lab Project

## Overview  
This is a Python machine learning project designed for students to learn and experiment with various ML algorithms.

## Getting Started  
To get started with this project, clone the repository:
```bash
git clone https://github.com/FrankTheSssnake/ML_Lab-Project.git
```

## Requirements

Make sure you have the following libraries installed:
```
    numpy
    pandas
    scikit-learn
    matplotlib
```
You can install them using pip:
```bash
pip install -r requirements.txt
```

## Usage

You can run the main script using:
```bash
python main.py
```
